# Beaver Revolution: World Dam Order

Это Obsidian-хранилище для канона, дизайна и производственных заметок HOI4-мода **Beaver Revolution: World Dam Order**.

## Быстрый вход

- [[CHATGPT_CONTEXT]] — краткий контекст для работы ассистента.
- [[01_CANON]] — текущий канон мира.
- [[1950е_стартовая_ситуация]] — обзор новой глобальной рамки.
- [[05_CURRENT_TASKS]] — ближайшие задачи и открытые несостыковки.
- [[99_DECISIONS_LOG]] — журнал принятых решений.
- [[Канада]] — первая страна вертикального среза.
- [[Данцигская Бобриная Автономия]] — главный бобриный очаг из нового документа.
- [[HOI4_структура_мода]] — техническая структура папки `hoi4_mod/`.

## Рабочий принцип

Мод развивается через вертикальные срезы: сначала одна страна и минимальный набор механик, затем расширение мира. Текущий технический срез — [[Канада]] (`CAN`), но после импорта документа появился кандидат на более тематический первый срез — [[Данцигская Бобриная Автономия]].
