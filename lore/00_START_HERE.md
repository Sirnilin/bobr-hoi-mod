# World Dam Order

Это Obsidian-хранилище для канона, дизайна и производственных заметок HOI4-мода **World Dam Order**.

Главная рамка проекта: альтернативные 1950-е, где послевоенный порядок разваливается на странные идеологические режимы, технокульты, военные диктатуры и локальные восстания с собственной внутренней логикой.

## Быстрый вход

- [[CHATGPT_CONTEXT]] — краткий контекст для работы ассистента.
- [[01_CANON]] — текущий канон мира.
- [[1950е_стартовая_ситуация]] — обзор новой глобальной рамки.
- [[05_CURRENT_TASKS]] — ближайшие задачи и открытые несостыковки.
- [[99_DECISIONS_LOG]] — журнал принятых решений.
- [[Данцигская Бобриная Автономия]] — первый кандидат на playable slice.
- [[Прусское Кавалерийское Ханство Монголии]] — кавалерийско-чайный режим в Азии.
- [[Скоростная Автомобильная Республика Японии]] — автомобильный технонационализм.
- [[HOI4_структура_мода]] — техническая структура папки `hoi4_mod/`.

## Рабочий принцип

Мод развивается через playable slice: сначала одна страна и минимальный набор механик, затем расширение мира. Активный кандидат на первый срез — [[Данцигская Бобриная Автономия]].
