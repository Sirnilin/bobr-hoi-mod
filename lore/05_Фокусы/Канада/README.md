# Фокусы Канады

Страна: [[Канада]]

HOI4-тег: `CAN`

## Назначение

Эта папка описывает концепты национальных фокусов для первого вертикального среза. Реальные файлы HOI4 должны появляться в `hoi4_mod/common/national_focus/` только после утверждения структуры дерева.

## Черновые ветки

- `CAN_river_authority` — государственный контроль рек и плотин.
- `CAN_forest_compromise` — сделка с лесными регионами и промышленниками.
- `CAN_listen_to_the_lodges` — осторожный контакт с бобровыми ячейками.
- `CAN_crack_down_on_cells` — репрессии против подпольных ячеек.
- `CAN_dominion_stability` — сохранение доминионного баланса.
