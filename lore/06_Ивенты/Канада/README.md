# Ивенты Канады

Страна: [[Канада]]

HOI4-тег: `CAN`

## Назначение

Эта папка описывает сюжетные и системные ивенты для канадского вертикального среза. Реальные HOI4-ивенты должны появляться в `hoi4_mod/events/` после утверждения цепочек.

## Черновые цепочки

- `can_beaver_cells.1` — первые сообщения о координированных работах на речных участках.
- `can_beaver_cells.2` — федеральное расследование: инженеры, лесники и странные плотины.
- `can_beaver_cells.3` — публичный кризис после инфраструктурного инцидента.
- `can_beaver_cells.4` — выбор курса: переговоры, подавление или технократический контроль.

## Минимум для первого прототипа

- [ ] 3-5 ивентов.
- [ ] Один стартовый новостной или политический сигнал.
- [ ] Один выбор, меняющий идею или решение.
- [ ] Связь с фокусами из [[05_Фокусы/Канада/README|фокусов Канады]].
