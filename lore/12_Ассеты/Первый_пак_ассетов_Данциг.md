# Первый пак ассетов: Данциг

Связанная страна: [[Данцигская Бобриная Автономия]].

## Статус

Рекомендованный первый визуальный пак для первого playable slice через Данциг.

## Минимум для первого playable slice

### Флаги

- [ ] `DZG.tga` — флаг Данцигской Бобриной Автономии.
- [ ] `medium/DZG.tga` — средний флаг.
- [ ] `small/DZG.tga` — малый флаг.

Промпт-идея: бело-красная послевоенная восточноевропейская символика, голубые речные линии, золотой или тёмный бобёр, индустриальная плотина, без текста.

### Портреты

- [ ] `portrait_dzg_alexander_kurva.dds` — [[Александр-Курва]].
- [ ] `portrait_dzg_vladislav_pozdnyakov.dds` — [[Владислав Поздняков]].
- [ ] `portrait_dzg_vanyuk.dds` — [[Гражданин Ванюк]].

Промпт-идея для Александра: альтернативно-исторический лидер 1950-х, восточноевропейская военная куртка, инженерные значки, фанатичный взгляд, фон с картами рек и плотин, политический портрет, без текста.

Промпт-идея для Владислава: мрачный идеолог, строгий костюм, символика последнего берега, холодный взгляд, тень плотины на фоне, политический портрет 1950-х, без текста.

Промпт-идея для Ванюка: спокойный кооперативный философ, преподаватель музыки, скромный костюм, партитуры и речные карты на фоне, мягкий свет, политический портрет 1950-х, без текста.
